# 5. Tests calcul TRS / TRG / TRE

Tester calculationService.ts.

Cas obligatoires :

Cas 1 — Production simple :
- tO = 480 min
- tR = 420 min
- tF = 390 min
- quantité produite = 360 000
- quantité conforme = 358 000
- cadence = 1000 unités/min
Vérifier :
- tN = quantité produite / cadence
- tU = quantité conforme / cadence
- DO = tF / tR
- TP = tN / tF
- TQ = tU / tN
- TRS = tU / tR
- TRG = tU / tO
- TRE = tU / tT

Cas 2 — Journée nettoyage uniquement :
- nettoyage local 08:00–12:00
- nettoyage équipement 12:00–17:00
Vérifier :
- pas de produit requis
- pas de lot requis
- pas de quantité requise
- pas de TRS artificiellement mauvais
- statut = activité non productive planifiée

Cas 3 — Journée mixte :
- nettoyage planifié 08:00–10:00
- production 10:00–17:00
Vérifier :
- nettoyage déduit correctement du temps requis
- TRS calculé sur la vraie fenêtre productive

Cas 4 — Arrêt non planifié :
- production 08:00–17:00
- panne 60 min
Vérifier :
- impact DO
- notification générée si seuil dépassé

Cas 5 — Sous-performance :
- pas d’arrêt déclaré
- cadence réelle faible
Vérifier :
- impact TP
- insight “sous-performance cadence”

Cas 6 — Qualité :
- quantité produite > quantité conforme
- rebuts élevés
Vérifier :
- impact TQ
- alerte qualité si seuil dépassé

Cas 7 — Mensuel :
Vérifier que :
- TRS mensuel = somme tU / somme tR
- ne jamais faire moyenne simple des TRS journaliers

Cas 8 — Division par zéro :
- tR = 0
- tF = 0
- tN = 0
Vérifier :
- pas de crash
- affichage N/A avec message clair

Livrables :
- tests unitaires Vitest
- CALCULATION_QA_REPORT.md
