# Roadmap

## v0.1 - Existing MVP baseline

- React frontend for TRS/OEE tracking
- Express backend API
- PostgreSQL schema through Supabase
- Deployment documentation for Vercel, Railway, and Supabase
- Operator, supervisor, and admin role concepts

## v0.2 - Open-source readiness

- Add contribution guide
- Add MIT license
- Add issue backlog
- Clarify GMP validation posture
- Add safe fictional demo data guidance

## v0.3 - Calculation confidence

- Expand unit tests for TRS/OEE formulas
- Add edge-case tests for zero planned time, stops, rejects, and missing cadence
- Add calculation examples in documentation
- Add clear rounding rules

## v0.4 - Production review workflow

- Add daily production review dashboard
- Add downtime and loss Pareto view
- Add supervisor validation workflow
- Add exportable review summary

## v0.5 - AI-assisted maintenance and review

- Draft issue triage summaries for maintainers
- Draft production review summaries from fictional/demo data
- Draft downtime investigation questions
- Keep AI-generated outputs clearly marked as draft decision support

## Long-term direction

Build a credible open-source manufacturing operations system for pharmaceutical production teams, focused on TRS/OEE, batch visibility, and GMP-aware workflow discipline.
