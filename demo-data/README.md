# Demo Data

This folder contains fictional data for public documentation, tests, screenshots, and contributor onboarding.

The data is not taken from a real pharmaceutical site. It does not represent real products, real batch numbers, real equipment, real suppliers, real employees, or real production performance.

## Dataset

`fictional-trs-oee-sample.csv` contains small sample rows with:

- Date
- Line
- Shift
- Fictional product name
- Fictional batch number
- Opening time
- Planned stop time
- Unplanned downtime
- Standard cadence
- Produced quantity
- Conforming quantity
- Downtime category
- Review note

## Intended use

Use this dataset to:

- Explain TRS/OEE calculations
- Build screenshots
- Create onboarding examples
- Test CSV import/export features
- Demonstrate dashboards without confidential manufacturing data

## Data-integrity note

For real regulated use, a site would need approved master data, controlled calculation rules, access control, audit trails, backup/retention rules, and QA-approved validation. This public dataset is only for open-source development and learning.
